About Me

I graduated with a Bachelors in Mechanical Engineering from the University of Maryland, College Park. Ultimately, I decided to leave that field and pursue web devlopment. I found that I had a strong eager to learn coding and I continually enjoy the day-to-day problems I am faced with.

As of December 2019, I graduated from the Full-Stack Web Developer Bootcamp at George Washington University. Currently, I am creating new and exciting projects to enhance my coding skills. Below is a list of projects that I have completed throughout my web development career.

Adopt Me is a React Application for browsing and adopting pets. The user can browse from cats and dogs to horses and barnyard animals. All of the available breeds will then appear in a dropdown menu for the user to click on. After submitting the information, a list of pets will render onto the page for the user to view. Clicking on a pet will prompt a modal to appear for the user to adopt the pet.

Through this project, I learned to how to use hooks, modals and async functions. I also became more familiar with React components, reach router, and JSX. To make coding easier, I used Prettier, an opinionated code formatter, that will automatically fix indentations for me after every save.
